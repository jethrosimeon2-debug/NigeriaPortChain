```
📋 Create Port Booking
Any connected wallet can create a booking. Arrival and departure must be future dates. Berth must be free.
Must be future
Must be after arrival
✅ Confirm / Complete / Cancel Booking
Confirm and Complete require the port authority wallet. Cancel can be done by the booking creator or authority.
📦 Retrieve Booking Record
🔍 Check Berth Availability
```
```
📄 Issue Electronic Bill of Lading
Only the port authority wallet can issue a Bill of Lading. Link it to a registered vessel and booking.
🔁 Transfer Bill of Lading
The current eBOL holder or the port authority can transfer ownership to a new wallet address (e.g. consignee or bank).
🔄 Update Cargo Status
Only the port authority wallet can update cargo status.
🔍 Retrieve Bill of Lading
```
```
👤 Register a Seafarer
Only the port authority wallet (NIMASA/NPA) can register seafarers and their certificates.
🔍 Lookup Seafarer by ID
🔎 Find Seafarer by Certificate Number
🔄 Update Certificate Status
Only the port authority wallet can update a seafarer's certification status.
```
